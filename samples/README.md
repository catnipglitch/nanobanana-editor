# samples ディレクトリについて

各APIの動作確認用です。
単独で実行可能
api keyは.env より読み込み
ファイルはoutputフォルダへ出力


- gcp_samples: Vertex AI や Gradio を使った GCP 向けサンプル
- gemini_api_sample: Google Gemini / Imagen API を直接呼び出す Text-to-Image サンプル
- open_router_sample: OpenRouter 経由で Gemini 画像モデルを利用するサンプル
- rembg: rembg ライブラリで背景削除を行う画像処理サンプル
- sample_images: 各種サンプルコードで利用する入力画像
- vertex_ai_api_samples: Vertex AI Imagen 3 / 4 系モデルの API サンプル
